# SceneReady

**The actor's toolkit — everything you need to prep, rehearse, and nail the audition.**

SceneReady is a native iOS app built for working actors and students. It brings your entire audition prep workflow into one place: script annotation, character breakdowns, cold read practice, teleprompter, monologue library, rehearsal journaling, and more.

## Features

- **Sides Annotation** — Import scripts (PDF or text) and mark them up with beat changes, objectives, tactics, and notes
- **AI Scene Partner** — Run lines with a text-to-speech scene partner that reads the other character's dialogue
- **Character Breakdown** — Structured worksheets for objectives, obstacles, tactics, backstory, physicality, and voice
- **Cold Read Timer** — Timed cold-read sessions to sharpen sight-reading skills
- **Teleprompter** — Scrolling teleprompter with adjustable speed for self-tape sessions
- **Monologue Library** — Browse curated monologues or add your own from text or PDF
- **Rehearsal Journal** — Log rehearsal notes, breakthroughs, and session reflections
- **Daily Warm-Ups** — Voice, body, and imagination warm-up exercises
- **Self-Tape Toolkit** — Checklists and guidance for professional self-tape setups
- **Industry Glossary** — Searchable reference for acting and industry terminology

## Tech Stack

- **React Native** + **Expo** (Expo Router with file-based routing)
- **TypeScript**
- **Zustand** for state management
- **AsyncStorage** for local data persistence
- **Expo Speech** for text-to-speech
- **Lucide React Native** for iconography

## Privacy

SceneReady collects **no user data**. All scripts, notes, and breakdowns are stored locally on the device. See [Privacy Policy](docs/privacy.html).
